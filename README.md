# METAL BURST

ブラウザで今すぐ遊べる、ターン制ロボ砲撃バトル。
弾道・風・**破壊できる地形**を読んで、相手のロボを撃ち抜け。スマホ／PC対応・無料。

▶ **遊ぶ: https://tmk4men.github.io/metal-burst/**

（往年のスマホゲーム *Blastron* に着想を得たオリジナル作品です。名前・グラフィック・サウンドはすべて新規制作で、原作とは一切関係ありません。）

## あそびかた
- **照準 / 発射**：画面の**右半分**をドラッグして向きと強さを決め、指を離すと発射
- **移動**：画面の**左半分**がパッド。1ターンにつき **4秒間**、上下左右に自由移動（PCは WASD / 矢印キー）
- **武器切替**：下のカード、または数字キー **1〜4**
  - キャノン（標準）/ グレネード（跳ねる）/ クラスター（分裂）/ レールガン（直進・高威力）
- 風と地形は毎ターン・毎ステージで変化。相手のHPを0にすれば勝ち
- モード：**VS COM** / **2人対戦**

## 特徴
- ピクセル単位で**破壊される地形**（5種のランダムステージ）
- 風・弾道・吹き飛び・落下ダメージのある物理
- スマホのタッチ操作に最適化したツインパッド
- 追加インストール不要、1ファイルの HTML で動作

## 技術
- 単一の `index.html`（Canvas 2D / WebAudio、依存ライブラリなし）
- ホスティング：GitHub Pages
